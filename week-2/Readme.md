# SQL Practice Repository

A collection of SQL practice programs covering important MySQL concepts and functions.  
This repository is organized day-wise for easy learning, revision, and interview preparation.

Practice SQL date and time functions.

Functions:
- CURDATE()
- NOW()
- DATE()
- YEAR()
- MONTH()
- DAY()
- DATEDIFF()
- DATE_ADD()
- DATE_SUB()
- DATE_FORMAT()
